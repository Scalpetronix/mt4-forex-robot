# Alpha Compass Stars EA for MetaTrader 4

Alpha Compass Stars is an EX4 Expert Advisor for desktop MetaTrader 4.

It analyzes gold on M5, opens and closes trades automatically, and pauses during risky news.

## Download

Download the latest `.ex4` file from the Releases page:

https://github.com/Scalpetronix/mt4-forex-robot/releases

## Installation

1. Download `AlphaCompassStars.ex4`.
2. Open MetaTrader 4.
3. Go to `File → Open Data Folder`.
4. Open `MQL4 → Experts`.
5. Copy `AlphaCompassStars.ex4` into the `Experts` folder.
6. Restart MetaTrader 4.
7. Open `Navigator → Expert Advisors`.
8. Drag `AlphaCompassStars` onto a chart.
9. Enable AutoTrading.

## Recommended chart

- Symbol: XAUUSD / Gold
- Timeframe: M5
- Platform: MetaTrader 4 desktop

## Risk warning

Trading Forex and gold involves high risk. This Expert Advisor does not guarantee profit. Test it on a demo account before using it on a live account.
