## Installation Guide

Follow these steps to install the Alpha Compass Stars `.ex4` robot in MetaTrader 4:

1. Download `AlphaCompassStars.ex4` from the Releases page.
2. Open **MetaTrader 4**.
3. Click **File → Open Data Folder**.
4. Open the folder: `MQL4/Experts`
5. Copy `AlphaCompassStars.ex4` into the `Experts` folder.
6. Restart MetaTrader 4.
7. Open **Navigator → Expert Advisors**.
8. Find **AlphaCompassStars**.
9. Drag the EA onto an **XAUUSD M5** chart.
10. Enable **AutoTrading**.
